# SmartPrep 🧠 | AI-Powered Interview Coach

SmartPrep is a sophisticated web application designed to help job seekers master technical interviews. By leveraging Google's Gemini AI, the platform analyzes user resumes to identify skill gaps and provides a simulated interview environment with real-time scoring and feedback.

## 🚀 Key Features

* **Resume-to-Interview Pipeline:** Upload a PDF resume to receive a personalized "Skill Gap Analysis" and custom-tailored interview questions.
* **Real-time AI Evaluation:** Get instant feedback on your answers, including scores on technical depth, communication, and conciseness.
* **Dynamic Readiness Score:** Track your improvement over time with an interactive dashboard and performance analytics.
* **Printable Assessment Reports:** Export your interview performance and AI-generated feedback into a clean, professional PDF report.
* **Speech-to-Text Integration:** Practice your verbal communication skills using integrated voice recognition.
* **Desktop & Tablet Optimized:** A high-end UI designed specifically for productivity on larger screens.

## 🛠️ Tech Stack

**Frontend:**
* React.js (Vite)
* Tailwind CSS (UI/UX)
* Chart.js (Analytics)
* Lucide React (Iconography)

**Backend:**
* Node.js & Express
* MongoDB Atlas (Database)
* Google Gemini AI (LLM Integration)
* JWT (Authentication)
